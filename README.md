# Notes App

Basic clone of Google Keep built using React

# Requirements

- Node ^16.13
- npm

# To run app locally

1. Clone the repo
   `git clone git@github.com:ikarabulut/keeper-app.git`
2. Move to keeper directory
   `cd keeper-app`
3. Install dependencies
   `npm install`
4. Run app on localhost
   `npm run start`
